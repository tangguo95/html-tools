## 📦 SHA 哈希工具说明

SHA 哈希工具支持以下功能：

- **算法选择**：SHA-1、SHA-256、SHA-384、SHA-512
- **文本哈希**：输入任意文本计算哈希值
- **文件哈希**：选择文件计算文件哈希
- **输出格式**：
  - 标准十六进制格式
  - 大写输出选项
  - 冒号分隔格式（每 2 字符用冒号分隔）

### 使用场景

- 文件完整性校验
- 密码存储（建议使用 SHA-256 或更高）
- 数据指纹生成
- 数字签名验证

### 安全提示

⚠️ SHA-1 已被证明存在碰撞攻击风险，不推荐用于安全敏感场景。建议使用 SHA-256 或 SHA-512。
