## 📦 SHA 哈希工具说明

SHA 哈希工具支持以下功能：

- **算法选择**：SHA-1、SHA-256、SHA-384、SHA-512
- **文本哈希**：输入任意文本计算哈希值
- **文件哈希**：选择文件计算文件哈希
- **输出格式**：
  - 标准十六进制格式
  - 大写输出选项
  - 冒号分隔格式（每 2 字符用冒号分隔）

### 使用场景

- 文件完整性校验
- 密码存储（建议使用 SHA-256 或更高）
- 数据指纹生成
- 数字签名验证

### 安全提示

⚠️ SHA-1 已被证明存在碰撞攻击风险，不推荐用于安全敏感场景。建议使用 SHA-256 或 SHA-512。

## 📄 大小写转换工具说明

大小写转换工具支持以下 7 种转换模式：

- **小驼峰 (camelCase)**：第一个单词小写，后续单词首字母大写
- **大驼峰 (PascalCase)**：每个单词首字母大写
- **蛇形 (snake_case)**：所有单词小写，用下划线分隔
- **短横线 (kebab-case)**：所有单词小写，用短横线分隔
- **全大写 (SNAKE_CASE)**：所有单词大写，用下划线分隔
- **平铺小写 (lowercase)**：所有单词小写，用空格分隔
- **全小写 (lowercase)**：所有单词小写，直接拼接

### 使用场景

- 变量命名规范统一
- API 参数格式转换
- SQL 字段名格式化
- 配置项命名标准化
- 代码风格统一

### 技术亮点

- ✅ 自动识别输入格式（支持 camelCase、PascalCase、snake_case、kebab-case 等）
- ✅ 支持中英文混合文本
- ✅ 智能处理数字和特殊字符
- ✅ 纯前端实现，无需服务器

## 📄 IP 地址计算器工具说明

IP 地址计算器支持以下功能：

- **子网计算**：输入 IP/CIDR 计算网络地址、广播地址、可用 IP 范围、主机数量
- **IP 范围计算**：计算起始 IP 到结束 IP 之间的 IP 数量
- **子网掩码转换**：子网掩码与 CIDR 前缀互转
- **多元输出**：支持二进制显示、通配掩码计算

### 使用场景

- 网络规划与设计
- IP 地址分配管理
- 子网划分计算
- 网络安全审计

### 功能特点

- ✅ 纯前端实现，无需服务器
- ✅ 支持 CIDR 表示法（如 192.168.1.0/24）
- ✅ 自动计算网络地址、广播地址、可用 IP 范围
- ✅ 支持通配掩码（Wildcard Mask）计算
- ✅ 支持二进制格式显示
- ✅ 支持 IP 范围计算和子网掩码转换

## 🔄 单位转换器工具说明

单位转换器支持以下 5 大类单位转换：

### 使用场景

- 日常生活单位换算
- 科学计算和单位转换
- 工程设计和国际单位制转换
- 烹饪配方单位调整

### 功能特点
