# HTML工具箱

一个纯HTML/CSS/JS实现的在线工具集合，无需后端服务，可离线使用。

## 工具列表

| 工具 | 说明 | 依赖 |
|------|------|------|
| 密码生成器 | 生成安全随机密码 | 无 |
| 文本比对 | LCS算法比对文本差异 | 无 |
| JSON格式化 | 格式化/压缩/验证JSON | 无 |
| Base64编码 | Base64编解码 | 无 |
| MD5加密 | 生成MD5哈希 | 无 |
| AES加密 | AES加解密 | Web Crypto API |
| 文件Base64 | 文件与Base64互转 | 无 |
| URL编解码 | URL编码/解码 | 无 |

## 使用方式

直接用浏览器打开 `toolbox.html` 即可使用。

## 功能详情

### 1. 密码生成器
- 自定义长度(4-128位)
- 字符类型：大写/小写/数字/符号
- 排除相似字符(0O, 1lI)
- 批量生成(最多20个)
- 密码强度评估

### 2. 文本比对
- LCS(最长公共子序列)算法
- 忽略大小写/空白字符
- 高亮显示差异

### 3. JSON格式化
- 格式化(2/4空格/Tab)
- 压缩
- 语法验证

### 4. Base64编码
- 编码/解码
- URL安全模式
- 支持UTF-8

### 5. MD5加密
- 纯JS实现，无依赖
- 32位/16位输出
- 大小写切换

### 6. AES加密
- 使用Web Crypto API
- 支持GCM/CBC/CTR模式
- PBKDF2密钥派生
- 随机密钥生成

### 7. 文件Base64
- 文件转Base64编码
- Base64转文件下载
- 支持任意文件类型

### 8. URL编解码
- 支持encodeURI/encodeURIComponent
- 编码/解码切换
